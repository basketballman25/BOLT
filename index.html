How to get song links that work:
• Dropbox: Upload MP3 → Share → copy link → change dl=0 to raw=1
• Google Drive: Upload MP3 → Share (anyone with link) → copy file ID → use: https://drive.google.com/uc?export=download&id=FILE_ID
• Direct MP3 URL: Any direct link ending in .mp3 works
